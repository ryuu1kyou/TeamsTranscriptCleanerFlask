ヘルプ

はじめに

Teams Transcript Cleaner をご利用いただき、ありがとうございます。このツールを使用して、Microsoft Teams の会議録を効率的に修正できます。

基本的な流れ
  1. Teams から会議録をエクスポート
  2. ファイルをアップロード
  3. 修正処理を実行
  4. 結果をダウンロード
トランスクリプトのアップロード

Microsoft Teams から会議録をテキストファイル（.txt）として保存し、アップロードします。

Teams からのエクスポート方法
  1. Teams で会議を開く
  2. 「レコーディングとトランスクリプト」を選択
  3. 「トランスクリプトをダウンロード」をクリック
  4. 保存されたファイルをアップロード
ファイルサイズの上限は 10MB です。
修正処理の実行

アップロードしたトランスクリプトに対して、様々な修正処理を実行できます。

処理モード
誤字脱字修正
音声認識エラーによる誤字脱字を修正します。最も基本的な処理です。
文法修正
文法的な間違いを修正し、より自然な日本語に改善します。
要約生成
会議の重要ポイントを抽出し、簡潔な要約を生成します。
カスタム処理
独自のプロンプトを使用して、特定の要求に応じた処理を行います。
AI モデルの選択
  • GPT-4o: 最新かつ最高性能のモデル（推奨）
  • GPT-4 Turbo: 高性能で高速なモデル
  • GPT-3.5 Turbo: コストパフォーマンスの良いモデル
ワードリスト機能

よく使用する修正パターンを辞書として保存し、効率的な修正を行えます。

ワードリストの作成
  1. 「ワードリスト」メニューから「作成」を選択
  2. 名前と説明を入力
  3. CSV形式で修正パターンを入力
  4. 保存して処理時に選択
CSV 形式の例
incorrect,correct
マイクロソフト,Microsoft
エクセル,Excel
チームス,Teams
トラブルシューティング
よくある問題
ファイルアップロードエラー
  • ファイル形式が .txt であることを確認
  • ファイルサイズが 10MB 以下であることを確認
  • ファイル名に特殊文字が含まれていないか確認
処理が失敗する
  • API 使用量の残高を確認
  • トランスクリプトの内容が適切か確認
  • 時間をおいて再試行
処理に時間がかかる
  • 文書の長さによって処理時間が変わります
  • 選択したモデルによって処理速度が異なります
  • ページを閉じても処理は継続されます
API 料金について

処理には OpenAI API の使用料金が発生します。

料金の仕組み
  • 使用したトークン数に基づいて計算
  • 選択するモデルによって単価が異なる
  • 処理前に概算コストが表示される
  • 使用量は管理画面で確認可能
注意: 長い文書や高性能モデルの使用は料金が高くなります。
